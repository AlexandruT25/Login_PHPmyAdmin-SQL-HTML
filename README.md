# Projeto Login de users registados em base de dados de PHPmyAdmin
Login de 5 users, podendo adicionar mais ou remover algum user, função realizada pelo Admin.

Tabela com informação do user logado, e os users registados na página de todos os users.
Todos os users podem  visitar uma página com automóveis, filtrando os veículos disponíveis com as respetivas características.
